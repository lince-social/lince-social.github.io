Código Aberto, Sem Fins Lucrativos, Sempre Gratuito

Lince é construído sobre princípios de transparência, propriedade comunitária e liberdade digital. Sem capital de risco, sem agendas ocultas — apenas software que serve você.

100% Gratuito & Código Aberto

Cada linha de código está aberta para inspeção, modificação e redistribuição. Sem planos premium, sem paywalls, sem limitações artificiais. O código-fonte completo está disponível sob uma licença permissiva.

Guiado pela Comunidade

O desenvolvimento é guiado pelas necessidades dos usuários, não por margens de lucro. Funcionalidades são priorizadas com base no feedback da comunidade e casos de uso reais. Todos podem contribuir com código, documentação ou ideias.

Privacidade por Design

Seus dados ficam no seu dispositivo. Sem contas na nuvem, sem telemetria, sem rastreamento. Lince funciona totalmente offline. Você é dono dos seus dados — exporte, faça backup, delete. Controle total.

O que é o Lince?

Em essência, Lince é um banco de dados pessoal para gerenciar qualquer coisa através da perspectiva de Necessidades e Contribuições.

Registro Pessoal

Crie um banco de dados SQLite local (seu 'DNA') que armazena tudo: tarefas, hábitos, inventários, projetos. Seu centro de comando pessoal, armazenado como um único arquivo portátil.

Necessidades & Contribuições

Um conceito simples mas poderoso: quantidades negativas são Necessidades (-2 Maçãs significa que você precisa de 2), positivas são Contribuições. Este modelo alimenta rastreamento, automação e trocas.

Automação Inteligente

Defina necessidades recorrentes, acione comandos do sistema com base em estados, automatize rotinas diárias. Do rastreamento de hábitos à automação do computador — se é previsível, Lince pode lidar.
